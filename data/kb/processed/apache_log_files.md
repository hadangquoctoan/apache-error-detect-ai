# Apache Log Files Reference

- **ErrorLog**: Logs server diagnostics and errors. Usually found at `/var/log/apache2/error.log`.
- **CustomLog**: Logs individual requests (Access Log). Usually at `/var/log/apache2/access.log`.

## Common Error Codes
- `[error]`: Serious issues that need attention.
- `[warn]`: Potentially problematic situations.
- `[notice]`: Normal but significant events.
- `[info]`: General information.
